# Перевод префиксной нотации в инфикную
## Установка зависимостей
### [cargo llvm-cov](https://lib.rs/crates/cargo-llvm-cov#readme-installation)
```bash
cargo +stable install cargo-llvm-cov --locked
```

## Запуск проекта
```bash
cargo run
```

## Запуск тестов
```bash
cargo test
```
### Покрытие
```bash
cargo llvm-cov
```
